1. Algorithm : Приготовить кашу
1. Отфильтровать воду;
2. Взять кастрюлю;
3. Налить воду в кастрюлю;
4. Включить плиту
Definition of Ready (DOR) : В кастрюле должна быть чистая вода;
Definition of Done (DOD): вода закипела;
4. Достать кашу;
5. Насыпать кашу в кастрюлю;
6. Достать ложку;
7. Помешать кашу
if (каша == густая)
{
 добавить еще воды 
}
else if (каша == жидкая)
{
ничего не делаем
}
8. Выключить плиту

{
выключить плиту
}
else
{
ничего не делаем, ждем пока закипит каша
}
Result: Каша готова
